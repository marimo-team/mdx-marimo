---
title: Fence contract
description: Supported marimo fence forms and options.
---

Use `python marimo`, `sql marimo`, or `markdown marimo` as the fence info string. The dotted form works too.

````mdx
```python.marimo
1 + 1
```
````

## Python cells

`echo=true` shows the source next to the output.

```python marimo echo=true
import marimo as mo

message = mo.md("**Python output** rendered through a marimo island.")
message
```

## Markdown cells

Markdown fences are converted to marimo markdown cells before extraction.

```markdown marimo echo=true
### Markdown island

This paragraph was written in a `markdown marimo` fence.
```

## Hidden setup

`include=false` runs a setup cell during extraction and leaves the rendered page focused on later outputs.

```python marimo include=false
hidden_value = "available to later cells"
```

```python marimo
mo.md(f"The setup value is `{hidden_value}`.")
```

## Static source fallback

`unparsable=true echo=true` shows source text for cells that should be displayed as author input.

```python marimo unparsable=true echo=true
this is author source, not executable Python
```
