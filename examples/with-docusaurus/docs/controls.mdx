# Controls

Each docs page can own its own marimo cells. This page uses a separate slider and dependent output.

```python marimo output=false
import marimo as mo
```

```python marimo
power = mo.ui.slider(1, 6, label="power")
power
```

```python marimo
mo.md(f"Two to the power of {power.value} is **{2 ** power.value}**.")
```
