# Summary

The Docusaurus sidebar links between MDX pages while each page keeps its own extracted marimo payloads.

```python marimo
import marimo as mo

rows = mo.ui.slider(1, 5, label="rows")
rows
```

```python marimo
items = "\n".join(f"- item {index}" for index in range(1, rows.value + 1))
mo.md(items)
```
