Each page below is a marimo notebook compiled into MDX by this package. The
site build includes its initial output, and the browser restores reactive
execution after the page loads.

## Learn marimo

- [Intro](intro.md) covers reactive execution, UI elements, Python cells, and
  the marimo editor.
- [Dataflow](dataflow.md) explains references, definitions, execution order,
  and reactive state.
- [UI](ui.md) exercises marimo's built-in inputs and form controls.
- [Markdown](markdown.md) combines Markdown, LaTeX, and interpolated Python
  values.

## Build interactive documents

- [Plots](plots.md) renders Matplotlib and interactive visualization output.
- [Layout](layout.md) composes rows, columns, accordions, tabs, and callouts.
- [SQL](sql.md) moves data between Python, SQL queries, and reactive outputs.

## Work with notebook formats

- [Python file format](fileformat.md) explains marimo notebook source and
  reusable notebook functions.
- [Markdown file format](markdown_format.md) covers Markdown notebook export,
  cell boundaries, and conversion back to Python.
- [For Jupyter users](for_jupyter_users.md) maps familiar notebook workflows
  to marimo's reactive execution model.
